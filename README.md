# 📊 Amazon Sales Dataset Analysis

## Overview
End-to-end data analysis of 50,000 Amazon sales orders 
using Python, Pandas, Seaborn and Matplotlib.

## Key Findings
- Beauty leads revenue (5.6M) but has lowest rating — churn risk!
- Middle East generates most revenue despite fewer orders — premium buyers
- Discounts have virtually zero impact on sales volume (1.67% variance)
- Only 0.59% order growth 2022→2023 — stagnant acquisition

## Tools Used
Python · Pandas · NumPy · Seaborn · Matplotlib

## Project Structure
- Data Cleaning (duplicates, datetime, outliers)
- Exploratory Data Analysis (EDA)
- Visualizations (6 charts)
